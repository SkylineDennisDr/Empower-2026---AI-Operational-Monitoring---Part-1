# Empower 2026 - AI Operational Monitoring - Part 1

This package deploys several elements used throughout the *AI Operational Monitoring* sessions at Empower 2026. In these sessions, the features *Pattern Matching*, *Behavioral Anomaly Detection*, *Proactive Cap Detection* and *Relational Anomaly Detection* are showcased using these elements.

The elements will be deployed under the view *DataMiner Catalog* > *Empower 2026*.

The elements deployed by this package will read in history data slowly and should therefore be installed before the sessions start. It takes 3 hours to read in data for the *Using Relational Anomaly Detection* session and 24h to read in data for the *Enhancing DataMiner AI with Human Feedback* session. 
This package should be used together with the companion package [Empower 2026 - AI Operational Monitoring - Part 2](https://catalog.dataminer.services/details/369b80b5-7112-427b-b697-249cbd9adace) , which loads some additional data in the elements created by this package.
